<h3 align="left">Installation</h3>
 
* Install required dependencies
  * ```dotenv```
  * ```express```
  * ```mysql2```
  * ```sequelize```
* Run ```mysql -u root -p``` in cmd or powershell terminal
* run ```npm seed``` to set seeds in db
* run ```npm start```
* open insomnia and connect to server routes
